# Кейс-задача № 4

## Задание

Известно общее количество голов драконьей стаи `N`, где `0 < N < 100`.
У одного дракона может быть максимум 7 голов.
Сила стаи определяется как произведение количества голов каждого дракона.

Необходимо определить максимально возможное значение силы стаи.

## Язык программирования

Python 3.

## Метод решения

Для решения используется динамическое программирование.

Пусть `dp[i]` — максимальная сила стаи при общем количестве голов `i`.

Для каждого значения `i` перебирается возможное количество голов у очередного дракона:
от 1 до 7. Затем выбирается вариант, при котором произведение максимально.

Формула перехода:

```text
dp[i] = max(h * dp[i - h])
```

где:

```text
1 <= h <= 7
h <= i
```

Начальное значение:

```text
dp[0] = 1
```

## Пример

Входные данные:

```text
10
```

Оптимальное разбиение:

```text
10 = 3 + 3 + 4
```

Выходные данные:

```text
36
```
