# Кейс-задача № 3

## Задание

Дан одномерный массив `A` размерности `N`. Необходимо найти сумму отрицательных элементов,
расположенных между максимальным и минимальным элементами массива.

## Язык программирования

Python 3.

## Описание алгоритма

1. Считывается размер массива `N`.
2. Считываются элементы массива.
3. Определяется индекс максимального элемента.
4. Определяется индекс минимального элемента.
5. Определяются левая и правая границы участка между максимумом и минимумом.
6. Выполняется проход по элементам между найденными границами.
7. Если элемент отрицательный, он добавляется к сумме.
8. Итоговая сумма выводится на экран.

## Пример

Входные данные:

```text
8
5 -1 -2 10 -3 -4 1 0
```

Выходные данные:

```text
-3
```
